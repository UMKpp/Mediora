# Mediora | Digital Healthcare Platform

Mediora is a modern digital healthcare platform designed to provide users with accessible healthcare services through a unified web application.

> This project is currently under active development.

## Overview

Mediora aims to simplify access to healthcare by offering tools that help users identify symptoms, locate nearby healthcare providers, and quickly access emergency medical services.

## Current Features

### Authentication

- User registration
- User login
- Google Sign-In integration planned
- Protected user routes
- Hidden admin access

### User Dashboard

- Personalized healthcare dashboard
- Accessibility-focused design
- Dark mode support
- Responsive user experience

### Symptom Checker

- Multi-step symptom selection
- Rule-based health guidance
- Risk level indication
- Health recommendations

### Doctor Locator

- Search doctors by symptoms
- Filter by specialization and location
- Sri Lanka district-based healthcare support

### Pharmacy Locator

- Search pharmacies by district
- Pharmacy filtering and recommendations

### Emergency Guidance

- Quick access to emergency healthcare services
- Suwa Seriya ambulance hotline integration
- Emergency contact information
- Major hospital emergency services

### Profile & Settings

- Personal information management
- Medical information management
- Accessibility preferences

## Getting Started

Clone the repository:

```bash
git clone https://github.com/UMKpp/mediora.git
```

Navigate to the project:

```bash
cd mediora
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
